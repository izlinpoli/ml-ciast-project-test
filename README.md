# What to include in your README for your machine learning project?
1. A title for your project(e.g. Titanic Survival Prediction App with Streamlit)
2. The purpose of your project (objective)
3. Description of your project structure (don't be too detailed on this!)
4. Describe the environment used by your project (What Python version? What packages are needed for this project) Usually you would have a requirement .txt file here
5. Description of your methodology (describe your Machine Learning (ML) workflow steps)
6. Display of results / method to access your result

Just keep in mind that the purpose of README is to let other users who never see your project before to understand your project well.
